# Legal Compliance Accountability Telegram Bot

Telegram bot that keeps compliance teams on track with deadline reminders, regulatory update digests, audit checklist nudges, and accountability check-ins. Commands include /deadline to set compliance milestone reminders, /update for weekly regulatory digest by jurisdiction, /checklist to run through SOC 2, GDPR, or HIPAA audit prep checklists interactively, /reminder for custom compliance task reminders, and /status for team compliance health summary. Integrates with Google Calendar and Notion via webhooks. Supports multiple jurisdictions: EU, US, UK, Canada, Australia. Built with python-telegram-bot, Docker-ready, self-hostable, MIT license.

## Setup
1. Set `TELEGRAM_BOT_TOKEN` in `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
